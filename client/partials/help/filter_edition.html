Filter Configuration


On this page you can see/edit your current filter settings.

Tip : You can get there through "Filter" menus in the Battle List and Statistics pages.

A filter is composed of a list of criterions. Each criterion can be enabled/disabled indepently.
In the image above, the "My Caster" criterion is enabled, while all other criterions are disabled.
You can enable/disable criterion by clicking on their label line.

To match the filter, a battle must match ALL active criterions (logical AND).

Disabled criterions settings are not forgotten, they're just, well, disabled when matching battles.

Note : the application remembers the filter state between sessions.

Date

You can see a select box for the matching criterion, and 3 input boxes for the reference date

You can filter battles by date relative to the reference date.

  • superior (>),
  • superior or equal (>=),
  • equal (==),
  • not equal (!=),
  • inferior or equal (<=),
  • inferior (>)

My Caster / Opp. Caster

You can see a select box for the matching criterion, a select box for the faction, and a multiple selection for the caster.

You can filter battles by your army's / your opponent's faction / caster.
If multiple casters are selected, the filter will match ANY of these (logical OR).
If no caster is selected, the filter will match the whole faction

Note : the multiple selection for the caster is empty until a faction is selected.

Opponent / Result / Scenario / Event

For all those criterion, you see a select box for the matching criterion, and a multiple selection for the value.

You can filter battles by your opponent's name / the result / the scenario / the event.
If multiple values are selected, the filter will match ANY of these (logical OR).
If no value is selected, the filter will match any battle (ie, it's useless).

Size

You see a select box for the matching criterion, and an input for the reference size.

You can filter battles by game size relative to the reference size.

  • superior (>),
  • superior or equal (>=),
  • equal (==),
  • not equal (!=),
  • inferior or equal (<=),
  • inferior (>)

Initiative

You see a select box for the matching criterion, a select box for the starting roll, and a select box for the first player.

You can filter battles by intitiative.
When both values are selected, the filter match battles with the exact same initiative (logical AND).
If no starting roll is selected, the filter match battle with any starting roll, and the selected first player.
Idem for the first player.

Tags

You see a select box for the matching criterion, and a multiple select box for the tag values.

You can filter battles by tags.

include any of
the filter match battles that include at least one of the selected tags.
include all of
the filter match battles that include all the selected tags (they can include other tags).
do not include all of
the filter match battles that do not include at least one of the selected tags.
include none of
the filter match battles that do not include any of the selected tags.
