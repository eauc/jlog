Battle Edition



This page allows full edition of a battle information.

Tip : You can get there by clicking on "Edit" button in the Battle View page,
or by clicking the "Add Battle" in the Battle List page.

Available actions :

Save
save/create battle and go back to Battle List page.
Close
discard changes and go back to Battle List page.
User Lists

When you see an input that looks like this :


it means the value can be selected amongst a list defined by the user (you).

The button on left allows you to enter a new value for the list.

The dropbox list in the middle allows the selection of a previously defined value.

The button on right allows you to delete the currently selected value from the list.

Note : lists are computed from the current battle list.
This means that values that are no longer referenced by any battle
(either after a battle edition or deletion) will automatically be removed from the list

Scenarios

The scenario list automatically includes Steam Roller scenarios names

Tags

Tags are user-defined labels that can be attached to any battle.

The app extract all tags names from the current battle list, and can use them to filter or compute statistics.
