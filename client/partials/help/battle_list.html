Battles List

The battles list display all your recorded battles.

Tip : You can get there by clicking on "Battle list" button in the Main Menu.

The last column display the result of the game :

Red
Defeat
Yellow
Draw
Green
Victory
A
by assassination
S
by scenario
C
by clock
T
by tiebreaker

Available actions :

<click on battle row>
display/edit battle information.
Add Battle
enter new battle.
Filter
You can filter the displayed battles according to multiple settings.
Sort By
You can choose the sort settings for the displayed battles.
Export
You can export the displayed battle list (ie after filtering and sorting) to various file formats.
Filter Action
Edit
Brings you to the Filter Settings Input page.
Invert
display battles that do NOT match the current filter.
Restore
undo "Invert" filter.
Enable
activate filter.
Disable
disactivate filter (the current filter settings are not lost, just disabled).

The filter button color change depending on the state of the filter :

white
filter inactive
green
filter active
blue
filter active and inverted
Sort Action

You can sort the displayed battle according to different criterions

The active sort is marked with a chevron

The default sort is by "Date, descending" (newest battles at the top).

Tip : You can revert the sort order by clicking on the active criterion. The chevron will change orientation.

sort ascending
sort descending
Export Action

You can export the currently displayed battles to different file formats.

BB Code
convert to internet forums table
CSV
Comma Separated Values text file, for use in spreadsheet editors
JSon
JSON text file, for use in other web applications

The export take into account filtering and sorting, so you can export only the set of battles you want.

Note : this is not the correct way to backup your battle list. Backup options are in the Backup page.
