Statistics



This page allows visualization of battles statistics.

Tip : You can get there by clicking on "Statistics" button in the Main Menu.

Available actions :

% <stats criterion>
display the stats type. You can compute various statistics about your battles.
<group by criterion>
display the stats groups. You can group the computed statistics by various values.
Filter
You can filter the battles used for computing the stats, according to multiple settings.
Statistics Type

You can choose which statistic to compute from your battles.
Basically, the app is able to count (almost) each of the informations available for battles
The default type is "Result".

Event
count references to each event (bar graph).
Initiative
count references to each initiative combination (pie chart).
My Caster/Opp. Caster
count references to each caster for all factions (bar graph).
My Faction/Opp. Faction
count references to each faction (bar graph).
Opp. Name
count references to each opponent name (bar graph).
Points
count the mean points scored by you and your opponents (balance graph).
Result
count your victories/defeats (pie chart).
Result Over Time
accumulate your victories/defeats over time (chart).
Scenario
count references to each scenario (bar graph).
Size
count references to each size (bar graph).
Tags
count references to each tag (bar graph).
Statistics Group By

You can choose to compute group statistics from your battles.
Basically, the app is able to group battles by (almost) each information available
The default group is "Total".

Event
group battles by event.
Initiative
group battle by each initiative combination.
My Caster/Opp. Caster
group battles by each caster (you must first select a faction a the top of the page).
My Faction/Opp. Faction
group battles by faction.
Opp. Name
group battle by opponent name (bar graph).
Result
group battles by game result.
Scenario
group battles by scenario.
Size
group battles by game size.
Tags
group battles by tag reference.
Total
one group, all battles.
Filter Action
Edit
Brings you to the Filter Settings Edition page.
Invert
use battles that do NOT match the current filter.
Restore
undo "Invert" filter.
Enable
activate filter.
Disable
disactivate filter (the current filter settings are not lost, just disabled).

The filter button color change depending on the state of the filter :

white
filter inactive
green
filter active
blue
filter active and inverted
